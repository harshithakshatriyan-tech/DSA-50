## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/java-dsa-practice.git
```

### 2. Navigate to the Project

```bash
cd java-dsa-practice
```

### 3. Compile and Run a Program

```bash
javac FileName.java
java FileName
```

👉 Example:

```bash
javac TwoSum.java
java TwoSum
```
